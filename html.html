Description de quelques balises les plus utilisées au début du cours developpent web


Toutes les definitions et les descriptions sont simpliées.
Voir le site < W3schools.com > pour plus de détails.





<Doctype> :
En HTML, le <doctype> est le préambule " <! DOCTYPE html> " requis en haut de tous les documents.
Son seul but est d'empêcher un navigateur de passer en soi-disant “quirks mode”
Le <doctype> n’est pas une balise HTML. C’est une instruction spécifique qui indique au
navigateur la version du langage HTML qui sera utilisée. Elle doit être utilisé au
tout début, avant la balise html.

En HTML5, la déclaration du Doctype a été simplifié, le Doctype n’est pas basé sur
le SGML, et ne nécessite donc pas une référence à une DTD!


<script> </script> :
L'élément HTML script est utilisé pour intégrer ou faire référence à un script exécutable.
Cela fait généralement référence à du code JavaScript mais ce peut également être un autre
type de script (par exemple WebGL).
Les scripts sans attribut async ou defer sont chargés et exécutés immédiatement avant que
le navigateur continue l'analyse de la page



<link> </link> :
L'élément HTML définit la relation entre le document courant et une ressource externe.
Cet élément peut être utilisé pour définir un lien vers une feuille de style, vers les icônes
utilisées en barre de titre ou comme icône d'application sur les appareils mobiles.



<i> </i> :
L'élément HTML i représente un morceau de texte qui se différencie du texte principal.
Cela peut par exemple être le cas pour des termes techniques, des phrases dans une langue
étrangère ou encore l'expression des pensées d'un personnage. Le contenu de cet élément est
généralement affiché en italique.
Exemple: texte en italique


<nav> </nav> :
L'élément HTML nav représente une section d'une page ayant des liens vers d'autres pages ou
des fragments de cette page. Autrement dit, c'est une section destinée à la navigation dans un
document (avec des menus, des tables des matières, des index, etc.).



<div> </div> :
L'élément HTML </div> (ou division) est le conteneur générique du contenu du flux. Il n'a aucun effet sur
le contenu ou la mise en page tant qu'il n'est pas mis en forme d'une manière quelconque à l'aide de CSS.
En tant que conteneur « pur », l'élément </div> ne représente rien en soi. Il est plutôt utilisé pour regrouper
le contenu afin qu'il puisse être facilement stylé à l'aide des attributs class ou id, pour marquer une section
d'un document comme étant écrite dans une langue différente (à l'aide de l'attribut lang), etc.

<body> </body> :
pour indiquer le corps de la page
Cet élément HTML représente le contenu principal du document HTML. Il ne peut y
avoir qu'un élément par document.


<head> </head> :
pour indiquer l'entête de la page
L'élément HTML fournit des informations générales (métadonnées) sur le document,
incluant son titre et des liens ou des définitions vers des scripts et feuilles de style.


<foot> </foot> :
La balise footer définit un pied de page d'un document ou une section du document.
Un élément footer doit contenir des informations sur l'élément parent. Un pied de page contient
généralement les informations sur l'auteur du document, de copyright, liens vers des termes d'utilisation,
les coordonnées de contact, etc..


<title> </title> : pour indiquer le tiltre de la page


<b> </b> :
Transforme le texte en caractères gras.
Cet élément HTML permet d'attirer l'attention du lecteur sur un contenu qui n'a pas,
pour autant, d'importance significative. Anciennement utilisé pour mettre le texte en gras.
Exemple: Texte gras.


<meta> </> :
La balise meta avec l'attribut http-equiv="refresh" est utilisée pour indiquer au navigateur
de rafraîchir ou de recharger la page et est généralement placée dans la section d'un document HTML.
<meta http-equiv="refresh" content="5"/> L'option content définit le temps en secondes pendant
lequel le navigateur doit attendre pour recharger la page.

Pour rediriger vers un autre site, l'option content doit être modifié pour inclure l'URL vers laquelle il doit
être redirigé et, comme dans l'exemple suivant, il redirigera immédiatement vers https://tutorialsgrey.com.

Exemple: <meta http-equiv="refresh" content="0;url=https://www.tutorialsgrey.com"/>
Cette commandande va rediriger votre site vers le site passer en url.


<strong> </strong> :
Indique une section de texte importante et la transforme en caractères gras.
Exemple: J'aime mon chien.


<br> : La balise <br> permet un retour à la ligne jusqu'à la marge gauche sans création
de nouveau paragraphe. Exemple:
une première ligne <br>
une deuxième ligne <br>.
