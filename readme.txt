Description & mode d'utilisation:

Le site est l'organisation de quelques exercice de devoir en html, javascript et css. Il compremd
5 pages: une page d'accueil, une page html, une page css et une page api web.

la page d'accueil est la page d'index.html qui comprend un message de bienvenu
et la barre de menu principal. 

Dans le menu, on peut naviguer pour visiter la page html.html qui comprend une quelques
balises les plus communes en html. Chaque balise a une definition et un exemple. 

Au bas de la page html.html on retrouve un tableau.

La page css.htm contient un tableau.

la page javascript est la page imc.htm, il contient deux formulaires qui demandent à l'utlisateur
d'entrer son taille et son poids pour calculer son indice de masse corporel (IMC). Un formulaire 
pour rentrer les données en kilogramme et en mètre et une autre formulaire pour rentrer les
données en livre et en pouce. Selon le formulaire choisi par l'utilisateur  la calculatrice 
retourne son imc en système métrique ou en système impérial.

La page api.html utilise "fetch" pour afficher des photos avec leurs identités. Une application
que je retrouve sur le tutoriel du site suivant:

 source code de API WEB: https://www.digitalocean.com/community/tutorials/js-fetch-api
